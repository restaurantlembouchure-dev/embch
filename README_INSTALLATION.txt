PACK COMPLET D'INSTALLATION - RESTAURANT OS V5

NOUVEAUTÉS V5
- Checklist différente par salarié, par jour et par service
- Écran admin avancé dans l'application
- Photo pour justifier :
  - les pertes
  - les ruptures
  - la validation finale
- Journal des photos dans l'onglet PHOTOS

FICHIERS
- Restaurant_OS_Complet_V5.xlsx
- APPS_SCRIPT.gs
- dossier public/

INSTALLATION
1. Importer Restaurant_OS_Complet_V5.xlsx dans Google Sheets.
2. Ouvrir Extensions > Apps Script.
3. Coller le contenu du fichier APPS_SCRIPT.gs.
4. Déployer en Web App avec accès : Toute personne disposant du lien.
5. Copier l'URL.
6. Ouvrir public/config.js et remplacer APPS_SCRIPT_URL par cette URL.
7. Mettre le dossier public/ en ligne (GitHub Pages, Netlify ou autre).

UTILISATION
- La checklist affichée dépend de :
  - l'employé connecté
  - le jour détecté automatiquement
  - le service détecté automatiquement
- Admin voit un panneau KPI.
- Pour une photo, saisir :
  - un nom de fichier
  - ou un lien
  Exemple : perte_accras_2026-03-30.jpg

INSTALLATION IOS
1. Ouvrir l'URL publique dans Safari.
2. Partager > Ajouter à l'écran d'accueil.

INSTALLATION ANDROID
1. Ouvrir l'URL publique dans Chrome.
2. Ajouter à l'écran d'accueil.
