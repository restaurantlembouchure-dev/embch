INSTALLATION RAPIDE

1. Remplacer APPS_SCRIPT_URL dans config.js par ton URL Apps Script (/exec)
2. Upload tous les fichiers sur GitHub (à la racine)
3. Activer GitHub Pages (main / root)
4. Ouvrir le lien sur Safari (iPhone)
5. Ajouter à l’écran d’accueil

Google Sheets requis :
- STOCK_PRODUITS (colonne A = Produits)
- CHECKLISTS (Employé | Jour | Service | Tâche)
- RUPTURES
- PERTES
